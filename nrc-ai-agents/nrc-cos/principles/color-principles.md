# NRC Color Principles

## Purpose

Visual Language, Typography, and Spacing establish how hierarchy, language, and relationship are made legible through structure alone. This document explains how color should behave once that structure already exists — never as the thing that creates meaning, only as the thing that can confirm it. Color has no independent standing in NRC's work. It exists only in service of a hierarchy that would already be true without it.

## Color Confirms Meaning, It Does Not Create It

If removing all color from a composition would make its hierarchy illegible or its meaning collapse, color was doing the job that structure — scale, position, weight — was supposed to do, and that is a failure, not an efficiency. Color should only ever confirm a relationship or an importance that already exists; where several colors appear together, the relationship between them should describe something true about the things they mark, not simply be a pleasing combination chosen on its own terms.

## Color Must Be Earned Like Any Other Emphasis

Every instance of color used to signal meaning needs a reason distinct from visual interest. Color must also never make something appear more certain, more urgent, or more important than it actually is — using it that way is the same failure as a claim the underlying work cannot support, expressed through hue instead of through words.

## Color Used Everywhere Has Stopped Meaning Anything

A composition saturated with color, applied broadly and without distinction, cannot use color to signal anything specific — there is nothing left for any one instance of it to stand out against. Restraint is what preserves color's capacity to mean something at all. Held in reserve, a single deliberate use of color is legible as a decision. Used constantly, it becomes wallpaper.

## The Same Color Use Should Mean the Same Thing, Every Time

Once a use of color has been established to signal a particular kind of meaning, it must signal that same meaning every time it recurs. A viewer who learns, correctly, what a given use of color indicates should never be forced to relearn it because the same signal was used differently somewhere else.

## Restraint Makes Color Legible as a Choice

Color's rarity is what allows its presence to be read as deliberate. Where color appears often, its appearance stops being a signal and starts being an ambient condition — present, but no longer communicating anything about the specific moment it occurs in. Sparing use is not a limitation on color. It is what allows color to keep functioning as color, rather than decay into texture.

## Emotional Influence Must Be Honest, Not Manufactured

Color can genuinely shape how something feels — this is a real and legitimate use of it. What color must never do is manufacture a feeling the content itself has not earned: urgency that isn't warranted, excitement about something ordinary, importance the substance underneath cannot support. Where color and content disagree about how significant something is, the content is right, and the color should be corrected to match it — never the reverse.

## Relationship to Visual Language, Typography, and Spacing

Visual Language, Typography Principles, and Spacing Principles each establish how hierarchy becomes legible without color's involvement. This document assumes that legibility already exists before color enters the composition at all. Nothing here should be read as license to use color to solve a hierarchy problem that structure has failed to solve on its own — that failure belongs upstream, and color is not a repair for it.

## Relationship to Later Principles

Specific palettes, hex values, contrast ratios, and accessibility requirements belong to `accessibility.md` and future brand and implementation documents, which will translate these principles into concrete values appropriate to a given medium and audience. None of those documents should establish a reason for color to exist beyond confirming meaning already present — only the specific values through which that confirmation happens.
