# NRC Representation Principles

## Purpose

Every principle established elsewhere in this layer governs meaning built from structure, language, or color. This document governs meaning built from an image — a photograph, an illustration, a render, a generated picture, a diagram, a single frame of film. However that image was produced, it makes a claim about what is real, and that claim is either honest or it isn't. This document is not about artistic style. It is about what representation owes to the truth it depicts.

## An Image Must Earn Its Place Like Any Other Decision

Every use of representation needs a reason distinct from visual interest. If a page or a moment would communicate the same understanding without the image, the image was standing in for an idea rather than supporting one. Representation is not exempt from the same test any other addition has to pass.

## Representation Must Show What Is True, Not What Is Convenient

An image should depict something real about the thing it represents — not a generic or convenient stand-in chosen because it was available, attractive, or easy to produce. However an image is made, and however abundant images become to produce, the obligation does not change: what is shown must correspond to what is actually true, not to what was quickest to reach for.

## An Image Must Never Claim More Than What Is True

Representation is one of the most powerful tools available for making something appear more significant, more finished, or more universal than the substance underneath actually is. Using it that way is the same failure as any unsupported claim, made in a different language. An image's confidence must never outpace the confidence the underlying work has earned.

## Emotional Influence Through Imagery Must Be Honest

An image can genuinely move someone — this is a real and legitimate use of representation. What it must never do is manufacture a feeling the underlying substance has not earned: aspiration, urgency, belonging that the work itself has not actually created. Where an image and the truth disagree about how something should feel, the truth is right, and the image should be changed to match it.

## Representation Should Clarify, Not Merely Decorate

An image included for atmosphere or visual interest alone, without helping someone understand something they would have understood less well through words or structure, has not earned its place. Beauty in representation, like beauty anywhere else in this language, is a consequence of an image being true and useful — never a separate goal pursued on its own.

## Restraint Applies to Representation Too

Imagery used constantly, or used simply because producing it has become cheap or abundant, loses its capacity to mean anything specific — the same failure already established for motion and color, applied here to pictures instead of movement or hue. Held in reserve, a single true image is legible as evidence. Used everywhere, imagery becomes wallpaper, regardless of how well any individual image was made.

## Relationship to Visual Language and Color Principles

Visual Language establishes that visual weight must correspond to real significance. Color Principles establishes that emotional influence must be honest, never manufactured. This document applies both to a different kind of visual claim — the claim an image makes simply by existing, regardless of how it was produced. None of these three documents should be read as governing separate territory; each governs the same underlying commitment to truthful communication, applied to a different visual mechanism.

## Relationship to Later Frameworks

Specific styles, production methods, and technical standards for photography, illustration, and generated imagery belong to future Frameworks and implementation documents. Nothing in those documents should establish a reason for representation to exist beyond truthfully supporting meaning already present — only the specific practices through which that truthfulness gets produced in a given medium.
